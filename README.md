# ImageToText
A tool for Genshin Impact Ugc.

Convert image to rich text and write to *.gil file.
# Usage
	ImageToHex <input file> <output file> <image file> <resolution> [extra options]
	  <input file>   The .gil project file to read.
	  <output file>  The .gil project file to write.
	  <image file>   The image file to convert to text.
	  <resolution>   The resolution of input image.
	Extra options: 
	  -pos=x,y   Set screen location.
	  -h=i   Set text box height.
	  -g=i   Set pixel granularity.
	  -p=guid   Set parent group.
